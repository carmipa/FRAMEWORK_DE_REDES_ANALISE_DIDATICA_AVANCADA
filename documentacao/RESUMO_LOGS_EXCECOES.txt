📚 TABELA RÁPIDA: QUANDO USAR CADA NÍVEL
═══════════════════════════════════════════════════════════════════════════

DEBUG
  Quando: Desenvolvimento, variáveis internas
  Exemplo: "valor_x = 42"
  Alerta: ❌ Não

INFO
  Quando: Eventos normais, operações bem-sucedidas
  Exemplo: "dns_cache hit", "login realizado"
  Alerta: ❌ Não

WARNING
  Quando: Problema ESPERADO (ex: DNS timeout)
  Exemplo: "domínio não encontrado", "cache expirado"
  Alerta: ⚠️  Talvez

ERROR
  Quando: Problema INESPERADO (ex: banco down)
  Exemplo: "falha de conexão", "erro de parsing"
  Alerta: 🔴 Sim

CRITICAL
  Quando: Sistema parado, não há operação possível
  Exemplo: "banco de dados totalmente indisponível"
  Alerta: 🛑 CRÍTICO
