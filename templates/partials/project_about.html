ℹ️Sobre o Projeto

Este framework foi desenvolvido como ferramenta didática para estudantes de Redes e Cyber Defense, permitindo calcular, validar e explicar conceitos de IPv4, CIDR, máscaras, wildcard, RFC1918, IPv6, comandos Cisco CLI e também consultar catálogos técnicos de Portas e Protocolos com DataGrid interativo (filtros, ordenação, paginação e exportação). Inclui a página Resolução de Problemas: VLSM para N localidades, topologia WAN (anel ou malha), prefixo WAN independente do CIDR da rede base, inferência opcional do CIDR da base pelo IP (modelo didático) e exportação em laboratório (.txt / .zip) ou relatório único para entrega (documentacao_cenario_rede.txt).

Competências demonstradas: Python Flask Redes TCP/IP IPv4 / IPv6 CIDR Portas e Protocolos DataGrid / UX Docker/VPS Deploy com HTTPS Cyber Defense
