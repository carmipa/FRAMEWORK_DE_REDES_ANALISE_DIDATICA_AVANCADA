ℹ️Sobre o Projeto

Este framework foi desenvolvido como ferramenta didática para estudantes de Redes e Cyber Defense, permitindo calcular, validar e explicar conceitos de IPv4, CIDR, máscaras, wildcard, RFC1918, IPv6, comandos Cisco CLI e também consultar catálogos técnicos de Portas e Protocolos com DataGrid interativo (filtros, ordenação, paginação e exportação).

Competências demonstradas: Python Flask Redes TCP/IP IPv4 / IPv6 CIDR Portas e Protocolos DataGrid / UX Docker/VPS Deploy com HTTPS Cyber Defense
